# My-minimal-portfolio-website
Welcome to my portfolio website!
This project is a simple and beginner-friendly two-page site that includes a Home page and a Contact page. It is built using HTML and CSS to create a clean, minimal, and easy-to-understand design.

**Pages Included**
1. Home Page

Displays a brief introduction

Includes a welcome message

Serves as the landing page for visitors

2. Contact Page

Contains a simple contact section

Provides ways to get in touch with me (email, social links, etc.)

**Technologies Used:**

HTML — Structure of the website

CSS — Basic styling for layout and design

**How to View the Project**

Download the project files

Locate the index.html file

Double-click it

It will open in your default web browser

No extra setup or installation is required.
